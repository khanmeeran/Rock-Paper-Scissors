# Rock Paper Scissors

## About

This is a simple JavaScript game where the user plays Rock, Paper, Scissors against the computer.

## Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Draw

## How It Works

1. User selects a choice (rock, paper, or scissors)
2. Computer randomly selects a choice
3. Both choices are compared
4. Winner is displayed in console

## Technology

- JavaScript

## Example Output

User Choice: rock  
Computer Choice: scissors  
Result: User wins!

## How to Run

1. Save file as `rock-paper-scissors.js`
2. Open terminal
3. Run:

node rock-paper-scissors.js
